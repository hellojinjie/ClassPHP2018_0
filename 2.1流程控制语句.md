# 流程控制语句

## 1. if ...
```
if (判断条件) {
  代码块;
}
```
 ## 2. if ... else ...
```
if (判断条件) {
  代码块;
} else {
  代码块;
}
```
## 3. if ... else if ... 
```
if (判断条件) {
  代码块;
} else if (判断条件) {
  代码块;
}
```
## 4. switch ... case ...
```
int num = rand(1, 4);
switch (num) {
  case 1:
    echo "hello" . 1;
    break;
  case 2:
    echo "hello" . 2;
    break;
  default:
    echo "hello". 3;
}
```
## 5. for 
```
for (初始化语句; 循环条件; 操作表达式) {
  代码块;
}
```
## 6. while
```
// 这是一个死循环
while (true) {
  代码块;
}
```
## 7. do while
```
do {
  代码块;
} while (循环条件);
```
