# 流程控制语句

## 1. if ...
```
if (判断条件) {
  代码块;
}
```
 ## 2. if ... else ...
```
if (判断条件) {
  代码块;
} else {
  代码块;
}
```
## 3. if ... else if ... 
```
if (判断条件) {
  代码块;
} else if (判断条件) {
  代码块;
}
```
## 4. switch ... case ...
```
switch () {
  case 1:
    break;
  case 
}
```
