# 流程控制语句

## 1. if ...
```
if (判断条件) {
  代码块;
}
```
 ## 2. if ... else ...
```
if (判断条件) {
  代码块;
} else {
  代码块;
}
```
## 3. if ... else if ... 
```
if (判断条件) {
  代码块;
} else if (判断条件) {
  代码块;
}
```
## 4. switch ... case ...
```
int num = rand(1, 4);
switch (num) {
  case 1:
    echo "hello" . 1;
    break;
  case 2:
    echo "hello" . 2;
    break;
  default:
    echo "hello". 3;
}
```
